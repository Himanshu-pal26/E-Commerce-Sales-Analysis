# 🛒 E-Commerce Sales Analysis Dashboard

 📌 Project Overview
This project focuses on analyzing e-commerce sales data to uncover valuable business insights related to sales performance, customer behavior, product trends, and profitability.

The analysis was performed using SQL, Power BI, and Excel to transform raw sales data into meaningful visual insights and interactive dashboards.

🚀 Project Objectives

- Analyze sales and profit trends
- Identify top-performing products and regions
- Understand customer purchasing behavior
- Track important KPIs
- Build an interactive business dashboard

 🛠️ Tools & Technologies Used

- SQL
- Power BI
- Excel
- GitHub
